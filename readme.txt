You just have to input the meta asked on the Option > Easy Meta tab and the plugin will automatically generate meta tag information on every page of your WordPress links. It will always show even you change your theme as many as you want and as long as the plugin is active. have a little fun with it ! :-)

== Description ==

Help you input meta tag information for claiming reason or other purposes.

You just have to input the meta asked on the Option > Easy Meta tab and the plugin will automatically generate meta tag information on every page of your WordPress links. It will always show even you change your theme as many as you want and as long as the plugin is active. have a little fun with it ! :-)

Already fixed the bug, thanks to Dan :-)

== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Open up EasyIcon configuration tab in Option tab.
